Tại đây sẽ là phần liệt kê, giới thiệu các blogs mà các bạn đã đăng trên [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj). Ví dụ:

###  [Blog 1 - GIẢI MÃ APACHE CELEBORN TRÊN AMAZON EMR](3.1-Blog1/)
Bài viết này chia sẻ phương pháp sử dụng Apache Celeborn làm Dịch vụ xáo trộn từ xa (Remote Shuffle Service - RSS) trên Amazon EMR để tối ưu hóa chi phí (dùng EC2 Spot) và cải thiện hiệu năng xử lý dữ liệu lớn (Big Data) chạy Apache Spark.

###  [Blog 2 - XÂY DỰNG DASHBOARD QUẢN LÝ VÁ LỖI ĐA TÀI KHOẢN BẰNG KIRO](3.2-Blog2/)
Bài viết chia sẻ bài học thực tế về việc xây dựng hệ thống Serverless Dashboard quản lý vá lỗi (Patch Management) đa tài khoản AWS bảo mật tuyệt đối, áp dụng quy trình Driven-by-Specification kết hợp với công cụ AI Kiro.

###  [Blog 3 - TỪ MULTI-AZ SANG SINGLE-AZ – TỐI ƯU CHI PHÍ TRÊN AWS](3.3-Blog3/)
Bài viết chia sẻ trải nghiệm thực tế về việc tối ưu hóa chi phí cho đồ án Team Task Management trên AWS bằng cách chuyển đổi kiến trúc từ Multi-AZ sang Single-AZ, chuyển đổi frontend tĩnh sang Amazon S3/CloudFront và thay thế Cron Job bằng AWS Lambda/EventBridge.
