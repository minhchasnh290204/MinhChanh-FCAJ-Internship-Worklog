This section lists and introduces blogs that I and my peers have published on [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj).

### [Blog 1 - DECODING APACHE CELEBORN ON AMAZON EMR](3.1-Blog1/)
This article shares how to use Apache Celeborn as a Remote Shuffle Service (RSS) on Amazon EMR to optimize costs (using EC2 Spot) and improve performance for Big Data processing with Apache Spark.

### [Blog 2 - BUILDING A MULTI-ACCOUNT PATCH MANAGEMENT DASHBOARD WITH KIRO](3.2-Blog2/)
This article shares practical lessons on building a fully secure Serverless Dashboard for multi-account AWS Patch Management, applying a Driven-by-Specification workflow combined with the Kiro AI tool.

### [Blog 3 - FROM MULTI-AZ TO SINGLE-AZ – OPTIMIZING COSTS ON AWS](3.3-Blog3/)
This article shares real experience optimizing costs for the Team Task Management project on AWS by converting architecture from Multi-AZ to Single-AZ, moving static frontend to Amazon S3/CloudFront, and replacing Cron Jobs with AWS Lambda/EventBridge.
